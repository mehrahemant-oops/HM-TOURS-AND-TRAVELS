# HM-TOURS-AND-TRAVELS
HM Travel is a dynamic travel website developed using the Django framework during my internship. The website allows users to explore travel destinations and packages and submit booking or inquiry requests. The frontend is designed using Bootstrap to ensure a responsive and user-friendly interface.
# HM-TOURS-AND-TRAVELS

## Features
- Responsive UI using Bootstrap
- Travel Package Listings
- Destination Management
- Contact & Inquiry Form
- Django Admin Panel
- Secure Form Validation

## Technology Stack
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite / MySQL

## Developed During
Python Full Stack Development Internship at OJD Consulting Pvt. Ltd.
